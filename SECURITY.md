# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing security@scoutvision.com. We will respond as quickly as possible.

## SOC2 Readiness
- All sensitive data is encrypted in transit and at rest.
- Role-based access control is enforced throughout the application.
- Audit logs are maintained for all critical actions.
- Regular dependency and vulnerability scanning is performed.
- Infrastructure is deployed using secure cloud best practices.
